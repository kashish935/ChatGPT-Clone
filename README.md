# 🚀 ChatGPT Clone with Memory Support

Building a **ChatGPT-like AI system** with Long-Term & Short-Term Memory capabilities.

## ✨ Key Features

* 👤 Multi-user Authentication (Register & Login)
* 💬 Multiple Independent Chats per User
* 🧠 Long-Term Memory (remembers past conversations)
* ⚡ Short-Term Memory (recent context handling)
* 🔄 Real-time Communication (Socket.IO)
* 📦 Vector Database Integration (Pinecone)
* 🔎 RAG Architecture with Vector Addressing
* 🚀 Backend Optimization for scalability
* 📱 Responsive UI

## 📚 What This Project Strengthened

* Vector Databases & Semantic Search
* Retrieval-Augmented Generation (RAG)
* AI Memory Systems (LTM vs STM)
* Real-time WebSocket Architecture
* Efficient Data Retrieval & Backend Optimization
